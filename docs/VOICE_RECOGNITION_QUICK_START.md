# 🎤 语音识别功能快速开始

## 🚀 立即使用

### 1. 启动应用

```bash
# 启动前端
cd frontend
npm run dev
```

### 2. 访问页面

打开浏览器访问：http://localhost:5173

### 3. 使用语音输入

1. 填写旅行信息（城市、日期等）
2. 在"额外要求"输入框下方，点击 **🎤 语音输入** 按钮
3. 允许浏览器使用麦克风（首次使用）
4. 看到"开始录音，请说话..."提示后，开始说话
5. 说完后，识别结果会自动填入输入框

## 📱 使用示例

### 场景：填写旅行额外要求

**传统方式**（打字）：
```
我想去看升旗，需要无障碍设施，对海鲜过敏，
希望酒店靠近地铁站，喜欢安静的环境
```
⏱️ 耗时：约 30-40 秒

**语音输入**：
1. 点击 🎤 按钮
2. 说："我想去看升旗，需要无障碍设施，对海鲜过敏，希望酒店靠近地铁站，喜欢安静的环境"
3. 自动填充

⏱️ 耗时：约 10 秒  
✅ 准确率：95%+

## 🔐 权限设置

### 首次使用

1. 点击麦克风按钮
2. 浏览器弹出权限请求：
   ```
   localhost 想要使用您的麦克风
   [阻止] [允许]
   ```
3. 点击 **[允许]**
4. 开始使用

### 权限被拒绝？

如果不小心点了"阻止"，可以这样恢复：

**Chrome/Edge:**
1. 点击地址栏左侧的 🔒 图标
2. 找到"麦克风"权限
3. 选择"允许"
4. 刷新页面

**Safari:**
1. Safari → 偏好设置 → 网站
2. 找到"麦克风"
3. 允许 localhost

## 💡 使用技巧

### 1. 提高识别准确率

✅ **环境**
- 在安静的环境中使用
- 避免背景噪音

✅ **说话方式**
- 说话清晰、语速适中
- 使用标准普通话
- 避免方言和口音

✅ **设备**
- 使用高质量麦克风
- 确保麦克风正常工作

### 2. 多次录音追加

```
第一次：点击 🎤 → 说"我想去看升旗"
结果：我想去看升旗

第二次：点击 🎤 → 说"需要无障碍设施"
结果：我想去看升旗 需要无障碍设施

第三次：点击 🎤 → 说"对海鲜过敏"
结果：我想去看升旗 需要无障碍设施 对海鲜过敏
```

### 3. 编辑识别结果

识别结果填入后，可以：
- ✏️ 手动修改错误的文字
- ➕ 继续添加更多内容
- 🗑️ 删除不需要的部分

## ❓ 常见问题

### Q: 点击麦克风没反应？

**A**: 检查以下几点：
1. ✅ 使用 Chrome/Edge/Safari 浏览器
2. ✅ 已授予麦克风权限
3. ✅ 麦克风设备已连接
4. ✅ 在 HTTPS 或 localhost 环境

### Q: 识别不准确？

**A**: 尝试：
1. 说话更清晰
2. 语速放慢
3. 使用标准普通话
4. 在安静环境中使用

### Q: 如何取消录音？

**A**: 
- 浏览器会自动停止（约 5-10 秒无声音）
- 或者等待识别完成

### Q: 支持英文吗？

**A**: 
- 当前配置为中文识别
- 可以修改配置支持英文
- 详见技术文档
